Ex_day37_week8

This repository contains a set of daily exercises focused on building strong fundamentals in Python and Machine Learning.

The exercises cover:

Logical data generation and filtering using pure Python
Feature interaction and numerical analysis with NumPy
Data cleaning and feature engineering using Pandas
Data visualization and interpretation with Matplotlib
Supervised machine learning using Random Forest, including model evaluation and feature importance analysis
All tasks use synthetic data to strengthen understanding of data behavior, feature design, and model selection without relying on external datasets.
